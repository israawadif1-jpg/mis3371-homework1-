# \# Project Description

# 

# This project is a \*\*Patient Registration Form\*\* designed for a medical website.

# The purpose of this assignment is to demonstrate the use of:

# 

# \* HTML5 structure

# \* Web forms

# \* External CSS styling

# \* Basic JavaScript

# \* Form layout using tables

# \* Proper project documentation

# 

# The form collects \*\*demographic and medical information from a patient\*\* and simulates a registration process for a medical clinic.

# 

# When the user submits the form, they are redirected to a \*\*Thank You page\*\* confirming the submission.

#
